# Dumbbell Home Workout Program

A 4-day Upper/Lower split designed for home training with dumbbells only.

- **~10 working sets** per muscle group per week
- **2× frequency** — each muscle trained twice per week
- **Inline SVG illustrations** — no external image dependencies
- Built as a single `index.html` — zero build step

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Click Deploy — done!

Or from terminal:
```bash
npx vercel
```

## Structure

```
index.html    — the full workout page
vercel.json   — Vercel config (static site)
README.md     — this file
```
